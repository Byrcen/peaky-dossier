> 💡 这不只是个粉丝项目,也是一次**产品练习**:如何把"一部很长的剧"做成"让人愿意一直点下去的互动"。<br>
> 真正的难点不在代码,在 👉 [**设计取舍**](#-设计取舍给-pm-看的部分)。

## 🎬 这是什么

一个**零依赖、单文件 HTML** 的互动叙事:用一面 **卷宗红线板**(Campbell 督察的案卷)做导航中枢,点开任意线索即进入 **满屏电影感像素场景** 逐屏播放。全程 **中英双语**、芯片配乐、进度可续。

> 双击 `index.html` 就能玩 —— 没有构建、没有后端、没有 node_modules。

---

## ✨ 一眼亮点

- 🧵 **红线板即机制** —— 剧里那几条跨季暗线,在板上是"免费"的:点一条红线,横跨六季的相关人物全亮起来
- 🔍 **透镜导航** —— 点人物 / 暗线 / 季,各是一条**不同的短弧**,而不是把你扔进 37 幕长河
- 🗡️ **物件即交互** —— 不用翻页按钮,**点场景里一件发光的剧情物件**(剃刀帽 / 蓝宝石 / 冒烟的左轮…)进入下一幕
- 🎞️ **电影质感** —— 黑边构图 + 字幕运镜淡入 + 暗角晕影 + 胶片颗粒
- 🌳 **关系图谱** —— 谢尔比家族树 + 六大帮派势力图
- ✅ **可收集** —— 看完的人物 / 走完的暗线打勾,把"很长"变成"想集齐"

---

## 🎮 怎么玩

```
按下开始  →  卷宗红线板  →  挑一条"线"  →  满屏电影场景  →  点物件逐幕  →  回板再抽一条
```

三种**透镜**,每条都短、有头有尾、走完自动回板:

| 入口 | 你会看到 |
|---|---|
| 👤 **点人物** | 弹出他的档案(身份 / 卷入的暗线),「跟随 XX 的故事」只播他出场的几幕 |
| 🧵 **点暗线** | 「▶ 追这条线 · N 幕」沿因果链跨季直击 |
| 🗓️ **点某季** | 「▶ 看 SX · N 幕」一整章 |

## 🧠 设计取舍(给 PM 看的部分)

做这个东西,真正的难点不是写代码,是三次"它不好玩"之后的取舍:

<table>
<tr><th>问题</th><th>洞察</th><th>方案</th></tr>
<tr>
<td>横版长卷太"装饰",信息没层次</td>
<td>剧情主线本就是"<b>警方调查谢尔比</b>",框架可以即剧情</td>
<td>🗂️ 做成 <b>侦探红线板</b>;跨季暗线变成可点的红线</td>
</tr>
<tr>
<td>点谁都一样、还得读完 37 幕</td>
<td>没人想读完整本,大家想"<b>抽感兴趣的线</b>"</td>
<td>🔭 <b>透镜模型</b>:每个入口=一条短的、各不相同的、走完回板的线</td>
</tr>
<tr>
<td>"点按钮翻页"太普通</td>
<td>点对点冒险的乐趣在"<b>点世界里的东西</b>"</td>
<td>🗝️ <b>物件交互</b>:每幕一件贴合剧情的发光物件,点它=下一幕</td>
</tr>
</table>

## 🗂️ 内容与考据

- **37 场景**:序幕 + 35 个关键 beat + 终幕(为大电影《The Immortal Man》留好 `season:7` 接口)
- **5 条命运暗线**:🩸 尚格莱特血仇 · 💎 被诅咒的蓝宝石 · 🔫 Polly 的子弹 · ☎️ Billy Grade 的电话 · 👻 汤米的两片雾
- **39 个角色**,中英双语
- 📖 **逐条对剧核查**:六季各派研究 agent 交叉 Fandom wiki + 维基剧集页 + 多家复盘,修正了 7 处常见误记
- 🈶 **中文译名校准**:对照豆瓣演职员表 / 维基百科中文,统一到主流叫法

---

## 🛠️ 技术栈

| | |
|---|---|
| 形态 | 单文件 `index.html`,零依赖,纯原生 JS |
| 渲染 | 内联 SVG 像素场景 + 像素场景助手库(`bg/figS/fire/smoke/horse…`)|
| 数据 | `SCENES / CHARACTERS / THREADS / BOARD_PINS` 与展示解耦 |
| 音频 | Web Audio 程序化合成(无音频文件):分季氛围 pad + 物件音效 |
| 其他 | localStorage 持久化 · `prefers-reduced-motion` · 移动端响应式 · 内置 `__selftest()` 数据完整性自检 |

## 🚀 运行

```bash
# 方式一:最简单
双击 index.html

# 然后浏览器打开 http://localhost:4792
```
